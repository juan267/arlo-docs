Once you have forms created, this guide shows you how to complete them, handle drafts, and finalize responses correctly.

---

## Start a New Response

### From the forms list

1. Go to **Custom Forms**
2. Find the form you want to complete
3. Click **New Response** or the (+) button
4. The form opens ready to complete

---

## Complete the Fields

### Text fields

- Click on the field and type your answer
- Respect maximum length if configured
- Required fields have an indicator (*)

### Numeric fields

- Enter only numbers
- Use period (.) or comma (,) for decimals according to your regional setting
- System validates minimum and maximum ranges

### Selection fields

**Single selection:**
- Click on the desired option
- Only one option can be selected

**Multiple selection:**
- Click on all options that apply
- You can select none, one, or several

### Date and time fields

- Use the date/time picker
- Some fields may have restrictions (e.g., no future dates)

### Photo fields

1. Click **Add Photo**
2. Options:
   - Take photo with device camera
   - Select from gallery
3. Photo is attached to the field
4. You can delete it and add another if needed

### Signature fields

1. Click on the signature area
2. Draw your signature with finger or mouse
3. Click **Accept** to confirm
4. You can clear and retry if needed

### Location fields

1. Click **Capture Location**
2. Allow location access if browser requests it
3. Coordinates are captured automatically
4. You can see the location on a mini-map

---

## Conditional Fields

If the form has configured dependencies:

1. Complete initial fields
2. Dependent fields will appear based on your responses
3. If you change a response, dependent fields may hide

---

## Save as Draft

### When to use drafts?

- Long inspections you can't complete at once
- Need to verify information before finalizing
- Want to review with a supervisor before completing

### How to save draft

1. Complete the fields you can
2. Click **Save Draft**
3. Response is saved with "Draft" status
4. You can close and return later

### Resume a draft

1. Go to the form's response list
2. Find your draft (indicated with "Draft" label)
3. Click to open it
4. Continue completing
5. Save as draft again or finalize

:::note[Drafts and exports]
By default, drafts are NOT included in data exports. This avoids including incomplete information in reports.
:::

---

## Finalize Response

### Automatic validation

When you click **Complete** or **Finalize**:

1. System verifies all required fields
2. Validates ranges and formats
3. If there are errors, shows which fields need attention
4. If everything is correct, response is marked as completed

### After completing

- Response is locked (not editable)
- Completion date and time is recorded
- Who completed the form is recorded
- Response appears in exports

---

## Completed Responses

### Characteristics

- **Immutable**: Cannot be modified after completing
- **Auditable**: Have timestamp and user
- **Exportable**: Included in exports by default
- **Permanent**: Part of historical record

### View completed response

1. Open the response from the list
2. All fields are read-only
3. You can see all captured data and photos

### What if I need to correct something?

If you discover an error in a completed response:

1. **You cannot edit it directly**
2. Options:
   - Create a new corrected response
   - Document the correction with a note
   - Contact administrator for exceptional cases

## Offline Work

### Availability

Offline behavior depends on how you access Arlo:

| Platform | Offline Support |
|----------|-----------------|
| Mobile app | Yes |
| Web browser | Limited |

### How it works (mobile app)

1. Open the form while you have connection
2. Form downloads to device
3. Complete response without connection
4. Save as draft
5. When you recover connection, sync
