The Arlo **Dashboard** provides a consolidated view of your farm's most important metrics. Instead of navigating through each module, you can see the overall operation status in one place.

---

## Access the Dashboard

The dashboard is typically the first screen when entering a farm. You can also access it from:

1. Click the Arlo logo or **Home**
2. Select **Dashboard** in the sidebar

---

## Dashboard Structure

### Executive Summary

High-level view with key indicators across all modules showing:
- **Sanitation**: Routes completed, coverage percentage, overdue routes
- **Field Tasks**: Active cycle, tasks in progress, actual vs. budgeted cost
- **Supplies**: Inventory value, products below minimum
- **Harvest**: Total harvested, percentage of target

## Metrics by Module

### Sanitation

| Metric | Description |
|--------|-------------|
| **Routes completed** | Total inspections finished in period |
| **Coverage** | % of lots inspected at least once |
| **Overdue routes** | Inspections past due date |
| **Active alerts** | Conditions that exceeded thresholds |

### Field Tasks

| Metric | Description |
|--------|-------------|
| **Active cycle** | Current field cycle and dates |
| **Active tasks** | Number of tasks in progress |
| **Actual cost** | Accumulated spending in cycle |
| **Variance** | Difference vs. budget (%) |

### Supplies

|--------|-------------|
| **Active cycle** | Current harvest cycle |

## Dashboard Widgets

### Trend Chart
Evolution of key metrics over time.

### Heat Map
Spatial view of indicators by lot with color coding.

### Recent Alerts
List of latest triggered alerts requiring attention.

### Pending Tasks
Checklist of items needing action today.
