Como **supervisor de campo**, eres quien captura la realidad del terreno en Arlo. Tu trabajo asegura que los datos sean precisos, completos y oportunos para que la administración pueda tomar buenas decisiones.

---

## Tu Rol en Arlo

### Qué deberías hacer

- Ejecutar o coordinar inspecciones de sanidad
- Registrar logs de trabajo completado
- Verificar calidad del trabajo de contratistas
- Reportar problemas o anomalías encontradas
- Sincronizar datos oportunamente

### Qué NO necesitas hacer

- Crear formularios o configurar alertas
- Planificar ciclos o presupuestos
- Aprobar compras o pagos
- Generar reportes ejecutivos

Esas tareas corresponden al administrador.

---

## Trabajo Móvil

### Dispositivo recomendado

- Smartphone o tablet con cámara
- Android o iOS reciente
- GPS activo
- Conexión a internet (al menos para sincronizar)

### Antes de salir al campo

```
□ Verificar batería del dispositivo (> 50%)
□ Verificar conexión a internet
□ Descargar formularios/rutas para trabajo offline
□ Confirmar rutas/tareas asignadas para hoy
```

---

## Trabajo Offline

### Cómo funciona

1. **Con conexión**: Descarga formularios y rutas asignadas
2. **Sin conexión**: Completa inspecciones normalmente
3. **Al recuperar conexión**: Sincroniza los datos capturados

### Descargar para offline

```
Antes de ir al campo:
1. Abre la ruta asignada
2. Verifica que el formulario esté descargado
3. Los lotes con mapas también se descargan
4. Verás indicador "Disponible offline"
```

### Sincronización

```
Al volver:
1. Conecta a WiFi o datos móviles
2. Abre Arlo
3. Verifica que aparezca "Sincronizando..."
4. Espera confirmación "Datos sincronizados"
5. Si hay error, reintenta o reporta al admin
```

:::caution[No cierres la app durante sincronización]
Espera a que termine de sincronizar antes de cerrar. Si cierras antes, podrías perder datos o tener que reintentar.
:::

## Por Módulo

### Sanidad: Ejecutar Rutas

**Tu flujo de trabajo:**

```
1. INICIO DEL DÍA
   → Abrir Arlo
   → Ver rutas asignadas
   → Descargar formularios offline

2. EN CAMPO
   → Abrir la ruta
   → Ir al primer lote
   → Completar formulario para cada árbol/punto
   → Tomar fotos cuando sea necesario
   → Repetir para cada lote

3. FIN DEL DÍA
   → Verificar que todos los datos estén completos
   → Sincronizar datos
   → Confirmar que la ruta aparece como "Completada"
```

**Consejos para datos de calidad:**

| Hazlo | Evita |
|-------|-------|
| Contar con precisión | Estimar "a ojo" |
| Tomar fotos claras | Fotos borrosas o sin contexto |
| Registrar todo | Omitir hallazgos "menores" |
| Sincronizar diario | Acumular varios días |

### Labores: Registrar Logs

**Tu flujo de trabajo:**

```
1. AL COMPLETAR TRABAJO
   → Verificar unidades realmente completadas
   → Abrir el ciclo de campo
   → Encontrar la labor correspondiente
   → Agregar log con:
     - Fecha del trabajo
     - Unidades completadas
     - Notas si hay observaciones

2. VERIFICACIÓN
   → Confirmar que el progreso se actualizó
   → Sincronizar datos
```

**Qué registrar en notas:**

```
✓ "Lluvia interrumpió trabajo a las 2pm"
✓ "Sector B tiene árboles más difíciles"
✓ "Contratista solicitó material adicional"
✗ (nada) cuando todo fue normal
```

### Cosecha: Registrar Entradas

**Si operas la báscula:**

```
1. CADA VIAJE
   → Pesar vehículo cargado
   → Registrar en Arlo:
     - Vehículo
     - Lote de origen
     - Grupo de cosecha
     - Peso de báscula
   → Sistema calcula peso neto automáticamente

2. VERIFICACIONES
   → Peso vacío del vehículo correcto
   → Lote correcto
   → Grupo correcto
```

## Resolución de Problemas

### No puedo sincronizar

```
1. Verifica conexión a internet
2. Cierra y abre Arlo nuevamente
3. Intenta conectar a WiFi
4. Si persiste, reporta al administrador
5. NO borres la app (perderías datos locales)
```

### La ruta no aparece

```
1. ¿Estás conectado? Puede que no se haya descargado
2. ¿Es para hoy? Verifica la fecha
3. ¿Te la asignaron? Confirma con administrador
4. Intenta refrescar la lista
```

### El formulario no carga

```
1. Verifica conexión
2. Cierra y abre la ruta
3. Verifica que está descargado para offline
4. Reporta al administrador si persiste
```

### Me equivoqué en un dato

```
1. Si aún no sincronizaste: Edita directamente
2. Si ya sincronizaste:
   - Para errores menores: Reporta al admin para corrección
   - Para errores graves: Documenta y reporta inmediatamente
```

## Buenas Prácticas

### Datos de calidad

```
✓ Registra en el momento, no "después"
✓ Sé preciso en conteos y mediciones
✓ Toma fotos que muestren claramente el hallazgo
✓ Usa las notas para contexto importante
✓ Sincroniza al final de cada jornada
```

### Gestión del tiempo

```
✓ Planifica tu ruta antes de salir
✓ Agrupa lotes cercanos
✓ Carga dispositivo la noche anterior
✓ Lleva batería portátil si las jornadas son largas
```

### Comunicación

```
✓ Reporta problemas graves inmediatamente (llamada/mensaje)
✓ Los problemas menores van en las notas de Arlo
✓ Si no puedes completar una tarea, avisa antes de que sea tarde
```

## Fotos: Cuándo y Cómo

### Cuándo tomar fotos

| Situación | ¿Foto? | Por qué |
|-----------|--------|---------|
| Plaga sobre umbral | ✓ | Evidencia para decisión |
| Síntomas inusuales | ✓ | Diagnóstico posterior |
| Daño significativo | ✓ | Documentación |
| Condición normal | ✗ | No aporta información |
| Ya hay fotos similares | ✗ | Evita redundancia |

### Cómo tomar buenas fotos

```
✓ Enfoca claramente el sujeto
✓ Incluye contexto (parte de la planta visible)
✓ Buena iluminación (no contraluz)
✓ Estable (no borrosa)
✓ Cerca cuando sea necesario ver detalle
```

## Preguntas Frecuentes

### ¿Qué hago si encuentro algo urgente?

1. Registra en el sistema normalmente
2. Además, llama o envía mensaje al administrador
3. No esperes a sincronizar para avisar de algo crítico

### ¿Puedo usar mi celular personal?

Depende de la política de tu finca. Pregunta al administrador. Consideraciones:
- Arlo funciona en cualquier dispositivo
- Considera batería y almacenamiento
- Los datos de la finca quedan en el dispositivo

### ¿Qué pasa si pierdo mi dispositivo?

1. Reporta inmediatamente al administrador
2. El admin puede desactivar tu sesión remotamente
3. Los datos sincronizados están seguros en el servidor
4. Los datos no sincronizados podrían perderse

### ¿Necesito internet todo el tiempo?

No. Puedes trabajar offline la mayor parte del día. Solo necesitas conexión para:
- Descargar rutas/formularios
- Sincronizar datos capturados
- Recibir actualizaciones
